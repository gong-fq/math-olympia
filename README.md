\# 🎓 数学 AI 助教（Netlify 安全版）



这是一个基于 Netlify Functions 的教学型数学 AI 助教应用，

强调 \*\*思路引导、分步讲解与数学规范\*\*，而非简单给答案。



\---



\## ✨ 特点



\- ✅ 教学型 Prompt（不是聊天机器人）

\- ✅ 分层教学模式（提示 / 分步 / 完整）

\- ✅ 自动识别题型（代数 / 线代 / 概率 / 微积分）

\- ✅ LaTeX + MathJax 数学排版

\- ✅ 无前端密钥，安全可公网部署
